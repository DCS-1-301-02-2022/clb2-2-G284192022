### 　情報工学科とは？
####  コンピュータの基礎から、高度なプログラミングまで。情報化社会をさらに進化させる、IT業界の新たなリーダーを育成。
*２年次から**コース**を選択し、専門的な学びを深めます。*

   1. システムエンジニアリングコース
   1. インターネット＆セキュリティコース
   1. 知能情報コース
